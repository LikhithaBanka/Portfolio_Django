## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/LikhithaBanka/Django-Portfolio.git
```

2. Navigate to the project folder

```bash
cd Django-Portfolio
```

3. Install Django

```bash
pip install django
```

4. Apply migrations

```bash
python manage.py migrate
```

5. Run the development server

```bash
python manage.py runserver
```

6. Open your browser

```
http://127.0.0.1:8000/
```
